# CondoTrack Design System - Professional Color Palette

## Overview

Sistema de cores profissional baseado nos padrões WCAG 2025 e melhores práticas para SaaS/B2B.

## Primary Brand Colors

Azuis profundos que transmitem confiabilidade e profissionalismo:

- brand-500: #1E6FD6 - CTAs primários, links
- brand-700: #0D4399 - Estados ativos
- brand-900: #062554 - Headers, alto contraste

## Accent Colors

Teal/emerald para sucesso e ações positivas:

- accent-500: #0FA37F - Sucesso, confirmações
- accent-700: #096D55 - Texto de sucesso

## Neutral Colors

Slate grays para backgrounds e texto:

- neutral-50: #F8FAFC - Background principal
- neutral-100: #F1F5F9 - Cards
- neutral-200: #E2E8F0 - Bordas
- neutral-500: #64748B - Texto secundário
- neutral-700: #334155 - Texto principal
- neutral-900: #0F172A - Máximo contraste

## Status Colors

### Success
- success-500: #0FA37F (WCAG AA: 4.7:1)
- success-700: #096D55 (WCAG AAA: 7.8:1)

### Warning
- warning-500: #D97706 (WCAG AA: 6.1:1)
- warning-700: #92400E (WCAG AAA: 9.2:1)

### Danger
- danger-500: #DC2626 (WCAG AA: 5.9:1)
- danger-700: #991B1B (WCAG AAA: 8.1:1)

### Info
- info-500: #1E6FD6 (WCAG AA: 5.2:1)
- info-700: #0D4399 (WCAG AAA: 7.4:1)

## Semantic Aliases

- : Mapeia para brand colors

## Usage Guidelines

### Textos
- Texto principal: content-DEFAULT (#334155) - WCAG AAA
- Texto secundário: content-muted (#64748B) - WCAG AA
- Placeholders: content-subtle (#94A3B8)

### Botões
- Primário: bg-brand-500 hover:bg-brand-600 text-white
- Sucesso: bg-accent-500 hover:bg-accent-600 text-white
- Perigo: bg-danger-500 hover:bg-danger-600 text-white

### Backgrounds
- App: bg-surface-muted (neutral-50)
- Cards: bg-surface-DEFAULT (white)
- Headers: bg-brand-800 ou bg-brand-900

## WCAG Compliance

Todas as cores de texto atendem ou excedem WCAG AA (4.5:1).
Cores principais atendem WCAG AAA (7:1+) onde possível.
